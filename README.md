# 🐶🐱 개 vs 고양이 판독 웹 서비스 (Dog vs Cat Classifier)

이 프로젝트는 사전 학습된 딥러닝 모델(`best_model_xception.keras`)을 사용하여 사용자가 업로드한 사진이 개인지 고양이인지 판독하는 간단한 웹 애플리케이션입니다. **Flask** 웹 프레임워크와 **TensorFlow/Keras**를 기반으로 작성되었습니다.

## 🌟 주요 기능
- 웹 브라우저를 통한 간편한 이미지 업로드
- Keras Xception 기반 모델을 활용한 이미지 이진 분류(개/고양이)
- 예측 결과 및 정확도(확률) 시각적 제공

## 📂 프로젝트 구조

```text
my_project/
│
├── app.py                         # Flask 웹 서버 및 예측 로직이 포함된 메인 실행 파일
├── best_model_xception.keras      # Keras 모델 파일 (직접 추가해야 함)
├── README.md                      # 프로젝트 설명서
├── static/
│   └── uploads/                   # 사용자가 업로드한 이미지가 임시 저장되는 디렉토리
└── templates/
    └── index.html                 # 웹 브라우저에 렌더링될 사용자 인터페이스(HTML)

```

## 🛠 필수 프로그램 및 라이브러리 (Requirements)

이 프로젝트를 실행하기 위해 필요한 패키지입니다. (Python 3.x 환경 권장)

* `tensorflow` (>= 2.x)


## 🚀 설치 및 실행 방법 (How to Run)

1. **프로젝트 폴더 생성 및 코드 저장:**
위의 구조에 맞게 파일을 배치합니다.
2. **모델 위치 확인:**
사용할 `best_model_xception.keras` 모델 파일을 `app.py`와 같은 위치(프로젝트 최상단 폴더)에 배치해 주세요.
3. **웹 서버 실행:**
터미널(명령 프롬프트)을 열고 프로젝트 폴더로 이동한 뒤, 아래 명령어로 서버를 실행합니다.
```bash
python app.py

```


4. **웹 브라우저 접속:**
터미널에 `* Running on http://127.0.0.1:5000` 문구가 나타나면,
웹 브라우저 주소창에 `http://localhost:5000` 을 입력하여 접속합니다.
5. **사진 판독:**
화면 중앙의 '파일 선택' 버튼을 눌러 동물 사진을 업로드하고 '판독하기' 버튼을 클릭하면 판별 결과를 확인하실 수 있습니다.

## ⚠️ 참고 사항 (Notes)

* **입력 이미지 크기:** 모델은 `150x150` 크기의 RGB 이미지를 입력받도록 구성되어 있으며, `app.py` 내에서 이미지를 자동 스케일링(`target_size=(150, 150)`)합니다.
* **예측 라벨 관련 (매우 중요):** 현재 코드에는 이진 분류의 결과(0.5 기준)를 바탕으로 라벨링이 설정되어 있습니다 (`score > 0.5`이면 '개', 그 외는 '고양이').
만약 예측 결과가 반대로 나오는 경우(개 사진에 고양이라 판별), `app.py` 38번째 줄 부근 `if score > 0.5:`의 조건을 반대로 수정하거나 리턴값을 맞바꿔주세요.
* **임시 파일 초기화:** 업로드된 이미지 파일들은 `./static/uploads/`에 지속해서 저장되므로, 주기적으로 용량 관리를 위해 지워주는 것이 좋습니다.
